BMF-CSSの特徴

  • 軽量かつシンプルな設計
  • CSSのみ（JSやプリプロセッサなし）
  • ダークモード対応
  • モバイルフレンドリー（レスポンシブ）
  • アクセシビリティ考慮（focus-visible、aria-*対応）
  • セマンティック + ユーティリティのハイブリッド設計

クイックスタート

インストール

npm install @bmf-san/bmf-css

または

yarn add @bmf-san/bmf-css

使用方法

HTMLファイルに直接インポート:

<link rel="stylesheet" href="node_modules/@bmf-san/bmf-css/dist/bmf-css.min.css">

JavaScriptでインポート:

// React, Vue, Angularなど
import '@bmf-san/bmf-css/dist/bmf-css.min.css';

詳細な使用方法はインストールガイドをご覧ください。
