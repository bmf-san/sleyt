BMF-CSSとは

BMF-CSSは、シンプルで軽量、使いやすいCSSフレームワークです。モダンなWebアプリケーションやレスポンシブWebサイトを素早く構築するために必要な基本的なスタイルとコンポーネントを提供します。

特徴

  • 軽量：必要最小限のスタイルとユーティリティのみを提供
  • モダン：CSSカスタムプロパティを活用
  • レスポンシブ：モバイルファーストのアプローチ
  • ダークモード対応：自動/手動切り替え
  • アクセシビリティ：WAI-ARIAに準拠
  • カスタマイズ可能：変数を変更するだけで簡単にカスタマイズ

使い方

npmでインストール:

npm install bmf-css

または、直接CSSをリンク:

<link rel="stylesheet" href="path/to/bmf.min.css">

カラーシステム

BMF-CSSは、抽象化されたセマンティックトークンを提供し、一貫性のあるデザインシステムを構築します。すべてのトークンはCSSカスタムプロパティで定義され、簡単にカスタマイズできます。

抽象トークン分類

カテゴリ 用途
text-* 通常文字・ラベル・補足など
bg-* 背景、カード、オーバーレイなど

color-* ステータス、セマンティック色
shadow-* シャドウ（レイヤー階層）
state-* hover, focus, active, disabled
component-* 特定UI部品向け（必要な場合のみ）
chart-* データビジュアル（必要な場合）

ユーティリティクラス

スペーシング

.m-* - マージンのユーティリティ

.mt-*, .mr-*, .mb-*, .ml-* - 方向別マージン

.mx-*, .my-* - 水平/垂直マージン

.p-* - パディングのユーティリティ

.pt-*, .pr-*, .pb-*, .pl-* - 方向別パディング

.px-*, .py-* - 水平/垂直パディング

ディスプレイ

.block - ブロック要素として表示

.flex - フレックスコンテナとして表示

.grid - グリッドコンテナとして表示

.inline - インライン要素として表示

.hidden - 要素を非表示

レスポンシブユーティリティ

.sm:* - 640px以上の画面

.md:* - 768px以上の画面

.lg:* - 1024px以上の画面

.xl:* - 1280px以上の画面

その他のユーティリティ

.rounded, .rounded-lg, .rounded-full - 角丸

.shadow, .shadow-md, .shadow-lg - シャドウ

.opacity-*, .z-* - 不透明度、重ね順

.overflow-hidden, .overflow-scroll - オーバーフロー

.sr-only - スクリーンリーダー用

システム設定の検出

@media (prefers-color-scheme: dark) {
    /* ダークモード変数 */
}
