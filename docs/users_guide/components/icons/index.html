アイコン

BMF-CSSで使用できるアイコンコンポーネントの使い方とバリエーション

基本的な使い方

BMF-CSSのアイコンは、フォントファイルをベースにした実装で、CSSのクラス名を使って簡単に実装できます。アイコンは以下の2つの方法で使用できます：

<span class="icon-home"></span>
<span class="icon-settings"></span>
<span class="icon-add"></span>
<span class="bmf-icon bmf-icon-md icon-user"></span>

利用可能なアイコン

BMF-CSSには以下のアイコンが含まれています：

icon-logo
icon-menu
icon-add
icon-user
icon-home
icon-content
icon-settings
icon-calendar
icon-angle-left
icon-angle-right

サイズバリエーション

アイコンは5つのサイズバリエーションを提供しています：

<span class="bmf-icon-xs icon-home"></span> <!-- 12px -->
<span class="bmf-icon-sm icon-home"></span> <!-- 16px -->
<span class="bmf-icon icon-home"></span>    <!-- 20px (デフォルト) -->
<span class="bmf-icon-lg icon-home"></span> <!-- 24px -->
<span class="bmf-icon-xl icon-home"></span> <!-- 32px -->

カラーバリエーション

アイコンのカラーは以下のクラスで変更できます：

<span class="icon-primary icon-home"></span>
<span class="icon-secondary icon-home"></span>
<span class="icon-muted icon-home"></span>
<span class="icon-danger icon-home"></span>
<span class="icon-success icon-home"></span>
<span class="icon-warning icon-home"></span>

テキスト付きアイコン

アイコンとテキストを組み合わせる場合は、icon-with-textクラスを使用します：

ホーム ユーザー 設定 
<span class="icon-with-text icon-home">ホーム</span>
<span class="icon-with-text icon-user">ユーザー</span>
<span class="icon-with-text icon-settings">設定</span>

アイコンの位置

テキストの右側にアイコンを配置する場合は、icon-rightクラスを使用します：

次へ 
<span class="icon-with-text icon-right icon-angle-right">次へ</span>

アイコンだけのボタン

アイコンだけのボタンは、button-iconクラスを使用します：

<button class="button-icon icon-primary icon-add"></button>
<button class="button-icon icon-secondary icon-user"></button>
<button class="button-icon icon-muted icon-settings"></button>

アニメーション付きアイコン

アイコンに回転アニメーションを適用するには、icon-spinクラスを使用します：

<span class="icon-spin icon-settings"></span>

無効状態のアイコン

アイコンを無効状態にするには、親要素にdisabledクラスを追加します：

<span class="disabled icon-home"></span>

ボタンとアイコンの組み合わせ

ボタンにアイコンを追加する例：

<button class="btn btn-primary">
  <span class="btn-icon icon-add"></span>
  追加
</button>
<button class="btn btn-secondary">
  設定
  <span class="btn-icon icon-settings"></span>
</button>

実装の詳細

BMF-CSSのアイコンは、フォントファイルとして実装されており、以下の特徴があります：

  • アイコンフォントはWebフォントとして読み込まれ、データURIとして埋め込まれています
  • 各アイコンはCSSの::before擬似要素を使用して表示されます
  • アイコンのコードポイントはCSS変数として定義されています（例：--icon-home）
  • アクセシビリティの観点から、装飾的なアイコンにはaria-hidden="true"を追加することを推奨します
