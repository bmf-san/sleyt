カラートークン

BMF-CSSで使用されるカラーパレットとカラートークンの説明

概要

BMF-CSSは、一貫性のあるデザインとユーザーエクスペリエンスを提供するために、定義済みのカラートークンを使用しています。これらのトークンは、テーマやダークモードなどの色彩変更をシンプルに行えるようにCSSカスタムプロパティ（変数）として実装されています。

基本カラー

BMF-CSSは、以下の基本カラートークンを提供しています：

UI色

--background 背景色
--foreground 前景色（テキスト）
--primary プライマリカラー
--secondary セカンダリカラー
--accent アクセントカラー
--muted 淡色（補助情報用）

状態色

--success 成功状態
--info 情報状態
--warning 警告状態
--danger 危険状態

ダークモード

BMF-CSSは、自動的にダークモードをサポートしています。data-theme="dark"属性または.darkクラスが適用されると、以下のカラートークンが自動的に切り替わります：

:root[data-theme="dark"] {
  --background: #121212;
  --foreground: #ffffff;
  --muted: #a1a1aa;
  --border: #2e2e2e;
  --input: #333333;

  /* その他のダークモード用カラー設定 */
}

ダークモードサンプル

ダークモードでは、テキストや背景色が自動的に調整されます。

カラートークンの使用方法

色はCSSカスタムプロパティを通じて簡単に使用できます：

CSSでの使用方法：

.my-element {
  color: var(--foreground);
  background-color: var(--primary);
  border: 1px solid var(--border);
}

テーマの変更：

/* カスタムテーマの作成 */
:root[data-theme="custom"] {
  --primary: #6d28d9;
  --secondary: #4f46e5;
  --accent: #8b5cf6;
}

/* テーマの適用 */
document.documentElement.setAttribute('data-theme', 'custom');

カラーユーティリティクラス

BMF-CSSは、カラートークンを利用するための便利なユーティリティクラスも提供しています：

テキストカラー：

  • .text-foreground - 前景色（デフォルトテキスト色）
  • .text-primary - プライマリカラー
  • .text-secondary - セカンダリカラー
  • .text-accent - アクセントカラー
  • .text-muted - 淡色テキスト
  • .text-success - 成功状態のテキスト
  • .text-info - 情報状態のテキスト
  • .text-warning - 警告状態のテキスト
  • .text-danger - 危険状態のテキスト

背景色：

  • .bg-background - 背景色
  • .bg-primary - プライマリ背景色
  • .bg-secondary - セカンダリ背景色
  • .bg-accent - アクセント背景色
  • .bg-muted - 淡色背景
  • .bg-success - 成功状態の背景色
  • .bg-info - 情報状態の背景色
  • .bg-warning - 警告状態の背景色
  • .bg-danger - 危険状態の背景色

ボーダーカラー：

  • .border-primary - プライマリボーダー色
  • .border-secondary - セカンダリボーダー色
  • .border-accent - アクセントボーダー色
  • .border-muted - 淡色ボーダー

アクセシビリティに関する考慮事項

BMF-CSSのカラーパレットは、WCAGのアクセシビリティガイドラインを考慮して設計されています：

  • テキストと背景のコントラスト比は、可読性を確保するためにWCAG AA基準（4.5:1）を満たすよう設計されています。
  • 色だけで情報を伝えることは避け、常にテキストやアイコンなどの視覚的な手がかりも提供するようにしてください。
  • カラーブラインドフレンドリーなデザインを促進するため、特に赤と緑の組み合わせには注意してください。

カスタムカラーの定義

プロジェクト固有のカラーパレットを定義する方法：

/* カスタムカラートークンの定義 */
:root {
  --brand-blue: #0056b3;
  --brand-gold: #ffcc00;
  --brand-dark: #222222;

  /* 既存のトークンをオーバーライド */
  --primary: var(--brand-blue);
  --accent: var(--brand-gold);
}

または、テーマファイルを作成してインポートする：

/* theme-custom.css */
:root {
  --primary: #0056b3;
  --secondary: #6c757d;
  --accent: #ffcc00;
  /* 他のカスタム色の定義 */
}

/* メインのCSSファイルでインポート */
@import 'theme-custom.css';
