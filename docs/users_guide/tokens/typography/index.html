タイポグラフィトークン

BMF-CSSで使用されるフォントサイズ、ウェイト、行高さなどのタイポグラフィトークンの説明

概要

BMF-CSSは、一貫性のあるタイポグラフィシステムを提供するために、定義済みのタイポグラフィトークンを使用しています。これらのトークンは、フォントサイズ、フォントウェイト、行高さ、フォントファミリーなど、テキストの視覚的特性を制御するためのCSSカスタムプロパティ（変数）として実装されています。

フォントサイズ

BMF-CSSは、以下のフォントサイズトークンを提供しています：

トークン 値 サンプル
--font-xs 12px 小さなテキストサンプル
--font-sm 14px 小さめのテキストサンプル
--font-base 16px 基本テキストサンプル
--font-lg 18px 大きめのテキストサンプル
--font-xl 20px より大きなテキストサンプル
--font-2xl 24px 見出しレベルのテキストサンプル
--font-3xl 30px 大見出しレベルのテキストサンプル 
/* フォントサイズトークンの使用例 */
.title {
  font-size: var(--font-2xl);
}

.subtitle {
  font-size: var(--font-xl);
}

.body-text {
  font-size: var(--font-base);
}

.caption {
  font-size: var(--font-sm);
}

フォントウェイト

BMF-CSSは、以下のフォントウェイトトークンを提供しています：

トークン 値 サンプル
--font-weight-normal 400 通常のテキストウェイト
--font-weight-medium 500 中程度のテキストウェイト
--font-weight-bold 700 太字のテキストウェイト 
/* フォントウェイトトークンの使用例 */
.heading {
  font-weight: var(--font-weight-bold);
}

.subheading {
  font-weight: var(--font-weight-medium);
}

.paragraph {
  font-weight: var(--font-weight-normal);
}

行高さ

BMF-CSSは、以下の行高さトークンを提供しています：

トークン 値 適用
--line-height-tight 1.25 見出しや短いテキストに適しています
--line-height-normal 1.5 一般的な本文テキストに最適です
--line-height-relaxed 1.75 読みやすさを重視する長文に適しています

Tight (1.25)

これは行高さがtightに設定されたテキストの例です。行間が狭くなるため、見出しや短いテキストに適しています。ただし、長文の場合は読みにくくなる可能性があります。

Normal (1.5)

これは行高さがnormalに設定されたテキストの例です。一般的な本文テキストに最適で、読みやすさとスペースのバランスが取れています。多くのウェブサイトやアプリケーションで使用されている標準的な行高さです。

Relaxed (1.75)

これは行高さがrelaxedに設定されたテキストの例です。行間が広くなるため、長文や複雑な内容の文章で読みやすさを向上させます。ブログ記事や記事本文などで効果的です。 

/* 行高さトークンの使用例 */
h1, h2, h3 {
  line-height: var(--line-height-tight);
}

p, li {
  line-height: var(--line-height-normal);
}

.article-content {
  line-height: var(--line-height-relaxed);
}

フォントファミリー

BMF-CSSは、以下のフォントファミリートークンを提供しています：

トークン 値 適用
--font-family-base システムフォントスタック 本文テキスト用の主要フォント
--font-family-heading システムフォントスタック（より強調） 見出し用のフォント
--font-family-mono 等幅フォントスタック コードサンプルや等幅テキスト

Base Font

これは基本フォントファミリーを使用したテキストの例です。一般的なコンテンツや本文テキストに適しています。

Heading Font

これは見出しフォントファミリーを使用したテキストの例です。見出しやタイトルに適しています。

Monospace Font

これは等幅フォントファミリーを使用したテキストの例です。コードやプログラミング関連のコンテンツに適しています。 

/* フォントファミリートークンの使用例 */
body {
  font-family: var(--font-family-base);
}

h1, h2, h3, h4, h5, h6 {
  font-family: var(--font-family-heading);
}

code, pre {
  font-family: var(--font-family-mono);
}

レタースペーシング

BMF-CSSは、以下のレタースペーシングトークンを提供しています：

トークン 値 適用
--letter-spacing-tight -0.025em 文字間隔を狭くし、大きい見出しなどに適しています
--letter-spacing-normal 0 標準的な文字間隔で、通常のテキストに使用されます
--letter-spacing-wide 0.025em 文字間隔を広くし、小さいテキストや全て大文字のテキストに適しています

Tight (-0.025em)

文字間隔が狭いテキストの例です。大きな見出しに効果的です。

Normal (0)

標準的な文字間隔のテキストの例です。通常のテキストコンテンツに使用されます。

Wide (0.025em)

文字間隔が広いテキストの例です。小さいテキストや全て大文字のテキストに適しています。 

/* レタースペーシングトークンの使用例 */
h1 {
  letter-spacing: var(--letter-spacing-tight);
}

p {
  letter-spacing: var(--letter-spacing-normal);
}

.small-caps {
  text-transform: uppercase;
  letter-spacing: var(--letter-spacing-wide);
}

利用方法

タイポグラフィトークンは、CSSカスタムプロパティとして実装されているため、簡単に使用できます。以下に一般的な使用例を示します：

/* タイポグラフィトークンの使用例 */
.page-title {
  font-size: var(--font-3xl);
  font-weight: var(--font-weight-bold);
  line-height: var(--line-height-tight);
  font-family: var(--font-family-heading);
  letter-spacing: var(--letter-spacing-tight);
}

.article {
  font-size: var(--font-base);
  font-weight: var(--font-weight-normal);
  line-height: var(--line-height-relaxed);
  font-family: var(--font-family-base);
  letter-spacing: var(--letter-spacing-normal);
}

.code-block {
  font-family: var(--font-family-mono);
  font-size: var(--font-sm);
  line-height: var(--line-height-normal);
}

レスポンシブタイポグラフィ

BMF-CSSでは、メディアクエリを使用して画面サイズに応じたタイポグラフィの調整が可能です：

/* レスポンシブなタイポグラフィの例 */
:root {
  --font-base: 16px;
  --font-lg: 18px;
  --font-xl: 20px;
  --font-2xl: 24px;
  --font-3xl: 30px;
}

@media (max-width: 768px) {
  :root {
    --font-base: 15px;
    --font-lg: 17px;
    --font-xl: 19px;
    --font-2xl: 22px;
    --font-3xl: 26px;
  }
}

@media (max-width: 480px) {
  :root {
    --font-base: 14px;
    --font-lg: 16px;
    --font-xl: 18px;
    --font-2xl: 20px;
    --font-3xl: 24px;
  }
}

カスタマイズ

プロジェクトの要件に合わせてタイポグラフィトークンをカスタマイズする方法：

  1. CSSファイルのオーバーライド： 独自のCSSファイルでトークンを再定義します。

    /* custom-typography.css */
:root {
  --font-base: 18px;  /* 基本フォントサイズを18pxに変更 */
  --font-family-base: 'Open Sans', sans-serif;  /* フォントファミリーを変更 */
  --line-height-normal: 1.6;  /* 行高さを調整 */
}

  2. インライン変数の上書き： 特定の要素や範囲で変数を上書きします。

    .special-section {
  --font-base: 17px;  /* このセクション内でのみフォントサイズを調整 */
  --line-height-normal: 1.7;  /* このセクション内でのみ行高さを調整 */
}

  3. CSS変数の上書き（JavaScript）： 動的な変更が必要な場合はJavaScriptで変数を更新します。

    // フォントサイズを動的に変更する
document.documentElement.style.setProperty('--font-base', '17px');

// ユーザー設定に基づいてフォントサイズを調整する関数
function adjustFontSize(sizePreference) {
  let newSize;
  switch(sizePreference) {
    case 'small':
      newSize = '14px';
      break;
    case 'large':
      newSize = '18px';
      break;
    default:
      newSize = '16px';
  }
  document.documentElement.style.setProperty('--font-base', newSize);
}

ベストプラクティス

BMF-CSSのタイポグラフィトークンを効果的に使用するためのベストプラクティス：

  • 一貫性を保つため、カスタムフォントサイズを定義する代わりに、提供されているトークンを使用しましょう。
  • 見出しレベルに応じて適切なフォントサイズと行高さのトークンを組み合わせましょう。
  • スマートフォンやタブレットなど、さまざまな画面サイズでのタイポグラフィの表示を確認しましょう。
  • アクセシビリティを考慮し、十分なコントラストと読みやすさを確保しましょう。
  • 過度に多くのフォントサイズやスタイルを使用しないようにし、デザインの一貫性を維持しましょう。

関連トークン

タイポグラフィと組み合わせて使用すると効果的な関連トークン：

  • カラートークン - テキストカラーや背景色
  • スペーシングトークン - テキスト要素間のマージンやパディング
  • ブレークポイント - レスポンシブタイポグラフィの実装
