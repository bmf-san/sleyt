Installation

Getting started with BMF-CSS

Installation Options

There are several ways to include BMF-CSS in your project:

NPM Installation

Install BMF-CSS using NPM:

npm install @bmf-san/bmf-css

Yarn Installation

Install BMF-CSS using Yarn:

yarn add @bmf-san/bmf-css

CDN

Include BMF-CSS directly from a CDN:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@bmf-san/bmf-css/dist/bmf-css.min.css">

Direct Download

Download the CSS files directly and include in your project:

Download Latest Release

Usage

Include BMF-CSS in your HTML file:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Project</title>
  <link rel="stylesheet" href="node_modules/@bmf-san/bmf-css/dist/bmf-css.min.css">
</head>
<body>
  <!-- Your content here -->
</body>
</html>

Import BMF-CSS in your JavaScript (ES6):

// In your main JS file
import '@bmf-san/bmf-css/dist/bmf-css.min.css';

// Your JavaScript code

Import BMF-CSS in your JavaScript (CommonJS):

// In your main JS file
require('@bmf-san/bmf-css/dist/bmf-css.min.css');

// Your JavaScript code

Basic Template

Here's a basic template to get you started:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>BMF-CSS Starter Template</title>
  <link rel="stylesheet" href="path/to/bmf-css.min.css">
</head>
<body>
  <header class="header">
    <div class="container">
      <h1>Page Title</h1>
    </div>
  </header>

  <main class="container">
    <section>
      <h2>Section Title</h2>
      <p>This is a paragraph with some text.</p>

      <div class="card">
        <div class="card-header">Card Header</div>
        <div class="card-body">
          <h3 class="card-title">Card Title</h3>
          <p>Card content goes here.</p>
          <button class="btn btn-primary">Button</button>
        </div>
      </div>
    </section>
  </main>

  <footer class="footer">
    <div class="container">
      <p>© 2024 My Website</p>
    </div>
  </footer>
</body>
</html>
