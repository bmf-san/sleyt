Base

Base components are the foundation of the Sleyt framework. They include reset, variables, and themes, providing a consistent design foundation.

reset.css

Provides minimal CSS reset for browser compatibility. It initializes default margins, paddings, and box sizes.

themes.css

Supports theme switching between light/dark modes. Theme switching can be done via data attributes.

variables.css

Defines design tokens as CSS variables. These can be used across the project for consistent styling and theme switching.

Themes

In Sleyt, you can easily switch between themes using data attributes. Both light and dark modes are supported by default, and custom themes can also be created.

How to Use Themes

Use the HTML data attribute to switch themes:

<html data-theme="dark">
  ...
</html>

Example of dynamically switching themes with JavaScript: 

// Switch to dark mode
document.documentElement.setAttribute('data-theme', 'dark');

// Switch to light mode
document.documentElement.setAttribute('data-theme', 'light');

Light Mode

Data attribute: data-theme="light"

Dark Mode

Data attribute: data-theme="dark"

Creating Custom Themes

To create your own theme, override CSS variables: 

[data-theme="custom"] {
  --color-primary: #6c5ce7;
  --color-secondary: #00cec9;
  --color-background: #ffeaa7;
  --color-text: #2d3436;
  /* Other variables... */
}

Variables

Sleyt uses CSS variables (custom properties) to manage design tokens. This enables consistent styling and easy theme switching.

Main Variable Categories

  --color-secondary: #64748b;
  --color-success: #10b981;

Typography Variables

  --font-family-base: 'Helvetica Neue', sans-serif;
  --font-family-heading: 'Helvetica Neue', sans-serif;
  --font-size-base: 1rem;
  --font-weight-normal: 400;
  --font-weight-bold: 700;

Spacing Variables

:root {
  --spacing-1: 0.25rem;
  --spacing-2: 0.5rem;
  --spacing-3: 0.75rem;
  --spacing-4: 1rem;
  --spacing-6: 1.5rem;
  --spacing-8: 2rem;

Breakpoint Variables

:root {
  --breakpoint-sm: 640px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 1024px;
  --breakpoint-xl: 1280px;
  --breakpoint-2xl: 1536px;
}
How to Use Variables

Example of using variables in CSS:

.button {
  background-color: var(--color-primary);
  color: white;
  padding: var(--spacing-2) var(--spacing-4);
  border-radius: var(--border-radius-md);
  font-weight: var(--font-weight-bold);
}

Example of manipulating variables with JavaScript: 

// Get CSS variable value
const primaryColor = getComputedStyle(document.documentElement)
  .getPropertyValue('--color-primary');

// Set CSS variable value
document.documentElement.style.setProperty('--color-primary', '#0088ff');

text-xs: Extra small text (12px)

text-sm: Small text (14px)

text-base: Base text (16px)

text-lg: Large text (18px)

text-xl: Extra large text (20px)

text-2xl: 2x size text (24px)

text-3xl: 3x size text (30px)

text-4xl: 4x size text (36px)

text-5xl: 5x size text (48px)

Font Weights

font-light: Light weight (300)

font-normal: Normal weight (400)

font-medium: Medium weight (500)

font-semibold: Semi-bold weight (600)

font-bold: Bold weight (700)

Text Alignment

text-left: Left alignment

text-center: Center alignment

text-right: Right alignment

text-justify: Justified alignment

Line Height

leading-none: No leading (1)

leading-tight: Tight leading (1.25)

leading-snug: Slightly tight leading (1.375)

leading-normal: Normal leading (1.5)

leading-relaxed: Slightly loose leading (1.625)

leading-loose: Loose leading (2)

Text Color

text-primary: Primary text

text-secondary: Secondary text

text-disabled: Disabled text

text-primary-color: Primary color text

text-success: Success message

text-danger: Error message

text-warning: Warning message

text-info: Info message

Whitespace

whitespace-normal: Normal wrapping

whitespace-nowrap: No wrapping

whitespace-pre: Preserves format

whitespace-pre-line: Preserve line breaks

whitespace-pre-wrap: Preserve whitespace and breaks

Word Break

break-normal: Normal breaks

break-words: Break words

break-all: Break on any character
