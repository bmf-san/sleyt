Sleyt CSS

ミニマルでモダンなCSSフレームワーク

ベース

ベースコンポーネントはSleytフレームワークの基礎となる要素です。リセット、変数、テーマを含み、一貫性のあるデザイン基盤を提供します。

reset.css

ブラウザ間の互換性を確保するための最小限のCSSリセットを提供。デフォルトマージン、パディング、ボックスサイズなどを初期化します。

themes.css

ライト/ダークモードなど、異なるテーマの切り替えをサポート。データ属性によるテーマ切り替えが可能です。

variables.css

カラー、フォント、スペーシングなどのデザイントークンをCSS変数として定義。プロジェクト全体で一貫した値を使用できます。

テーマ

Sleytではデータ属性を使用して簡単にテーマを切り替えることができます。ライトモードとダークモードを標準でサポートしており、カスタムテーマも作成可能です。

テーマの使い方

HTMLのdata属性を使用してテーマを切り替えます：

<html data-theme="dark">
  ...
</html>

JavaScriptでテーマを動的に切り替える例：

// ダークモードに切り替え
document.documentElement.setAttribute('data-theme', 'dark');

// ライトモードに切り替え
document.documentElement.setAttribute('data-theme', 'light');

ライトモード

データ属性: data-theme="light"

ダークモード

データ属性: data-theme="dark"

カスタムテーマの作成

独自のテーマを作成するには、CSS変数を上書きします： 

[data-theme="custom"] {
  --color-primary: #6c5ce7;
  --color-secondary: #00cec9;
  --color-background: #ffeaa7;
  --color-text: #2d3436;
  /* その他の変数... */
}

変数

Sleytでは、CSS変数（カスタムプロパティ）を使用してデザイントークンを管理します。これにより一貫したスタイリングとテーマの切り替えが容易になります。

主要な変数カテゴリ

カラー変数

:root {
  --color-primary: #3b82f6;
  --color-secondary: #64748b;
  --color-success: #10b981;
  --color-danger: #ef4444;
  --color-warning: #f59e0b;
  --color-info: #3b82f6;
}

タイポグラフィ変数

:root {
  --font-family-base: 'Helvetica Neue', sans-serif;
  --font-family-heading: 'Helvetica Neue', sans-serif;
  --font-size-base: 1rem;
  --font-weight-normal: 400;
  --font-weight-bold: 700;
}

スペーシング変数

:root {
  --spacing-1: 0.25rem;
  --spacing-2: 0.5rem;
  --spacing-3: 0.75rem;
  --spacing-4: 1rem;
  --spacing-6: 1.5rem;
  --spacing-8: 2rem;
}

ブレイクポイント変数

:root {
  --breakpoint-sm: 640px;
  --breakpoint-md: 768px;
  --breakpoint-lg: 1024px;
  --breakpoint-xl: 1280px;
  --breakpoint-2xl: 1536px;
}
変数の使用方法

CSSで変数を使用する例：

.button {
  background-color: var(--color-primary);
  color: white;
  padding: var(--spacing-2) var(--spacing-4);
  border-radius: var(--border-radius-md);
  font-weight: var(--font-weight-bold);
}

JavaScriptで変数を操作する例：

// CSS変数の値を取得
const primaryColor = getComputedStyle(document.documentElement)
  .getPropertyValue('--color-primary');

// CSS変数の値を設定
document.documentElement.style.setProperty('--color-primary', '#0088ff');

コンテナ

コンテナはページコンテンツを適切な幅に制限し、画面サイズに応じて自動的に調整するレイアウト要素です。画面中央に配置され、レスポンシブデザインの基盤となります。

基本的なコンテナの使い方

基本的なコンテナクラスの使用例：

<div class="container">
  <!-- コンテンツがここに入ります -->
</div>

コンテナの中央揃えとマージンを自動で調整： 

<div class="container mx-auto">
  <!-- 左右のマージンが自動的に均等になります -->
</div>

コンテナのブレイクポイント別最大幅：

ブレイクポイント クラス 最大幅
デフォルト .container 100%
Small (sm) .container 640px
Medium (md) .container 768px
Large (lg) .container 1024px
Extra Large (xl) .container 1280px

グリッド

CSSグリッドは二次元のレイアウトシステムで、行と列の両方向に沿って要素を配置できます。複雑なレイアウトを少ないコードで実現できるパワフルなツールです。

グリッドレイアウトの基本

基本的なグリッドの例：

<div class="grid grid-cols-3 gap-4">
  <div>項目 1</div>
  <div>項目 2</div>
  <div>項目 3</div>
  <div>項目 4</div>
  <div>項目 5</div>
  <div>項目 6</div>
</div>

3列グリッドの例：

グリッドアイテムのスパン：

1（2列幅）

ユーティリティ

ユーティリティクラスは、特定のスタイルを迅速に適用するための小さなクラスです。これらは、特定のプロパティや値を持つ単一のクラスであり、迅速なスタイリングを可能にします。

borders.css

ボーダーのスタイル、幅、色を制御するユーティリティ。

colors.css

テキストカラー、背景色などのカラーユーティリティ。

display.css

要素の表示方法を制御するユーティリティ。

effects.css

要素に影やトランジションを追加するユーティリティ。

position.css

要素の位置を制御するユーティリティ。

spacing.css

マージンとパディングのユーティリティ。

typography.css

フォントサイズ、フォントウェイト、行間などのタイポグラフィユーティリティ。

タイポグラフィ

タイポグラフィユーティリティはテキストのスタイリングを制御します。フォントサイズ、フォントウェイト、行間などを設定できます。

タイポグラフィユーティリティの使用例

フォントサイズ

text-xs: 極小サイズのテキスト (12px)

text-sm: 小さいサイズのテキスト (14px)

text-base: 基本サイズのテキスト (16px)

text-lg: 大きいサイズのテキスト (18px)

text-xl: より大きいサイズのテキスト (20px)

text-2xl: 2倍サイズのテキスト (24px)

text-3xl: 3倍サイズのテキスト (30px)

text-4xl: 4倍サイズのテキスト (36px)

text-5xl: 5倍サイズのテキスト (48px)

フォントウェイト

font-light: 軽い太さ (300)

font-normal: 通常の太さ (400)

font-medium: 中程度の太さ (500)

font-semibold: やや太い (600)

font-bold: 太字 (700)

テキストの整列

text-left: 左揃え

text-center: 中央揃え

text-right: 右揃え

text-justify: 両端揃え

行の高さ

leading-none: 行間なし (1)

leading-tight: 詰まった行間 (1.25)

leading-snug: やや詰まった行間 (1.375)

leading-normal: 通常の行間 (1.5)

leading-relaxed: やや広い行間 (1.625)

leading-loose: 広い行間 (2)

テキストカラー

text-primary: 主要テキスト

text-secondary: 補助テキスト

text-disabled: 無効テキスト

text-primary-color: プライマリカラー

text-success: 成功メッセージ

text-danger: エラーメッセージ

text-warning: 警告メッセージ

text-info: 情報メッセージ

ホワイトスペース

whitespace-normal: 通常の折り返し

whitespace-nowrap: 折り返しなし

whitespace-pre: 整形済みテキスト

whitespace-pre-line: 行の折り返しを保持

whitespace-pre-wrap: スペースと改行を保持

改行制御

break-normal: 通常の改行

break-words: 単語の途中で改行

break-all: すべての文字で改行可能
