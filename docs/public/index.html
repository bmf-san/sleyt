Getting Started

Learn how to install and use Sleyt CSS Framework in your projects.

Quick Start: Install via npm or include directly in your HTML.

NPM Installation

npm install sleyt

Then import in your CSS:

@import 'sleyt/dist/css/index.css';

CDN Usage

<link rel="stylesheet" href="https://unpkg.com/sleyt@latest/dist/css/index.css">

Include in your HTML head section.

Glass Morphism

Create beautiful transparent effects with backdrop blur.

Glass Effect

Standard glass morphism effect with backdrop blur. 

<div class="glass">Content</div>

Light Glass

Lighter glass effect for subtle transparency. 

<div class="glass-light">Content</div>

Heavy Glass

Heavy glass effect with more opacity. 

<div class="glass-heavy">Content</div>
Available Glass Classes: .glass, .glass-light, .glass-heavy, .backdrop-blur
