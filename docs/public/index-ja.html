はじめに

Sleyt CSS Frameworkをプロジェクトにインストールして使用する方法を学びましょう。

クイックスタート: npmでインストールするか、HTMLに直接含めてください。

NPMインストール

npm install sleyt

その後CSSでインポート:

@import 'sleyt/dist/css/index.css';

CDN使用

<link rel="stylesheet" href="https://unpkg.com/sleyt@latest/dist/css/index.css">

HTMLのheadセクションに含めてください。

クイック例

Sleytへようこそ

ガラスモーフィズムデザインを持つモダンなCSSフレームワーク。 

<div class="card">
  <div class="card-header">
    <h4 class="font-semibold">Sleytへようこそ</h4>
    </div>
  <div class="card-body">
    <p class="mb-4">ガラスモーフィズムデザインを持つモダンなCSSフレームワーク。</p>
    <button class="btn btn-primary">はじめる</button>
  </div>
</div>

コンポーネント

ガラスモーフィズム効果を持つモダンなWebインターフェースの構築ブロック。

ボタン

様々なスタイルとサイズのインタラクティブ要素。 

<button class="btn btn-primary">プライマリー</button>

カード

コンテンツをグループ化するための柔軟なコンテナ。

サンプルカードコンテンツ

<div class="card">
  <div class="card-body">
    コンテンツ
  </div>
</div>

フォーム

入力フィールドとフォームコントロール。 

<input type="text" class="form-control">

アラート

ユーザーへの通知メッセージ。

成功メッセージ
警告メッセージ 
<div class="alert alert-success">成功</div>

バッジ

小さなステータス表示とラベル。

新着 アクティブ 5 
<span class="badge badge-primary">新着</span>

テーブル

構造化データの表示。

名前 ステータス
アイテム1 アクティブ 
<table class="table">...</table>

チャート

透明効果を持つ美しいデータ視覚化コンポーネント。

バーチャート

カスタマイズ可能な色で美しいバーチャートを作成。 

<div class="chart">
  <div class="bar-chart">
    <div class="bar-chart-item" data-variant="primary" data-height="md"></div>
  </div>
</div>

折れ線チャート

グラデーション塗りつぶしのスムーズな折れ線チャート。 

<div class="chart">
  <svg class="line-chart-svg">
    <path d="..." stroke="var(--color-primary)"/>
  </svg>
</div>

ドーナツチャート

凡例付きのエレガントなドーナツチャート。 

<div class="donut-chart">
  <svg class="donut-chart-svg">
    <circle cx="50" cy="50" r="30"/>
  </svg>
</div>

ガラスモーフィズム

バックドロップブラーで美しい透明効果を作成。

ガラス効果

バックドロップブラーを持つ標準的なガラスモーフィズム効果。 

<div class="glass">コンテンツ</div>

ライトガラス

微細な透明感のためのライトガラス効果。 

<div class="glass-light">コンテンツ</div>

ヘビーガラス

より多くの不透明度を持つヘビーガラス効果。 

<div class="glass-heavy">コンテンツ</div>
利用可能なガラスクラス: .glass, .glass-light, .glass-heavy, .backdrop-blur
