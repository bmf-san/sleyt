Icon Font Implementation

このサイトではSVGアイコンの代わりに、擬似要素を使ったアイコンフォントを実装しています。この方法により、アクセシビリティが向上し、メンテナンスも容易になります。

アイコンの例

ホームアイコン ユーザーアイコン 追加アイコン 設定アイコン コンテンツアイコン ロゴアイコン

利点:

  • アクセシビリティの向上: 擬似要素を使うと、スクリーンリーダーがアイコンを読み上げることがなく、テキストのみを読み上げます
  • 動的な変更に強い: HTMLを変更せずにCSSだけで見た目を変更できます
  • コードがすっきりする: 入れ子のiタグが不要になり、HTMLがすっきりします
  • 管理がしやすい: アイコンをすべてCSSで一元管理できます

アクセシビリティに関する注意事項

色は情報伝達の唯一の手段として使用するべきではありません。視覚障害を持つユーザーのためにテキストや形状など、他の視覚的な手がかりと組み合わせてください。

テキストと背景のコントラスト比：

  • WCAG AAレベル： 通常テキストで4.5:1以上、大きなテキストで3:1以上
  • WCAG AAAレベル： 通常テキストで7:1以上、大きなテキストで4.5:1以上

ユーティリティカラーは装飾的な要素やグラフィック表現に適していますが、テキスト表示の場合は適切なコントラストを確保してください。
